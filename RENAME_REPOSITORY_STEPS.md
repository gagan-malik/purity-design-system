# Rename Repository to purity-design-system

## üéØ Rename on GitHub

1. **Go to Repository Settings:**
   - Navigate to: https://github.com/gagan-malik/presto-design-system/settings

2. **Scroll to Repository Name Section:**
   - Find "Repository name" (near the top of the settings page)

3. **Rename the Repository:**
   - Change from: `presto-design-system`
   - Change to: `purity-design-system`
   - Click **"Rename"** button

4. **Confirm:**
   - GitHub will ask you to confirm
   - Type the new name to confirm
   - The repository will now be at: `https://github.com/gagan-malik/purity-design-system`

## üìù After Renaming

Once renamed on GitHub, run these commands locally:

```bash
# Update git remote URL
git remote set-url origin https://github.com/gagan-malik/purity-design-system.git

# Verify the change
git remote -v

# Test connection
git fetch origin
```

Then commit all the updated references!
